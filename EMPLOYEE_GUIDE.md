# 従業員向け利用手順

## 1. このツールでできること

- Eight から名刺データを取得する
- 取得した相手向けのメール文面を作る
- Outlook にメール下書きを自動作成する

## 2. 利用前に必要なもの

- 管理者から共有されたアプリ URL
- 自分のユーザー名
- 自分のパスワード
- 自分の Microsoft アカウント
- Eight にログインできる自分の PC

## 3. ログインする

1. アプリ URL を開く
2. 左側のサイドバーで自分のユーザー名とパスワードを入力する
3. `ログイン` を押す

## 4. Eight セッションを準備する

Eight のログイン情報は直接アプリに入力しません。
自分の PC でセッションファイルを作成してアップロードします。

### 4-1. `login_helper.py` を入手する

アプリの `Eight ログイン` ページから `login_helper.py` をダウンロードできます。

### 4-2. 自分の PC で実行する

自分の PC のターミナルで以下を実行してください。

```bash
pip install playwright playwright-stealth
playwright install chromium
python login_helper.py
```

### 4-3. Eight にログインする

ブラウザが開いたら、Eight にログインしてください。

- ID / パスワード入力
- 2段階認証

完了すると `eight_session.json` が作成されます。

## 5. Eight セッションをアップロードする

1. アプリの `Eight ログイン` ページを開く
2. `eight_session.json` をアップロードする
3. `セッション有効` と表示されれば完了

## 6. Outlook 認証を行う

1. アプリの `設定` ページを開く
2. `認証コードを発行` を押す
3. 表示された URL をブラウザで開く
4. 表示されたコードを入力する
5. 自分の Microsoft アカウントでログインする
6. アプリに戻って `認証完了を確認` を押す

## 7. 名刺データを取得する

1. `スクレイピング` ページを開く
2. 日付フィルタを選ぶ
3. `名刺データを取得` を押す

取得した連絡先は画面下部に表示されます。

## 8. テンプレートを確認・編集する

1. `設定` ページを開く
2. テンプレートを選ぶ
3. 必要に応じて件名・本文を編集する
4. 保存する

## 9. Outlook 下書きを作成する

1. `下書き作成` ページを開く
2. 下書きを作りたい連絡先を選ぶ
3. テンプレートを選ぶ
4. `下書き作成` を押す

完了後、Outlook の下書きフォルダにメールが作成されます。

## 10. よくあるトラブル

### Eight のセッションが期限切れ

- `Eight ログイン` ページで期限切れと表示された場合は、もう一度 `login_helper.py` を実行して新しい `eight_session.json` をアップロードしてください。

### Outlook が未認証

- `設定` ページで再度認証してください。

### 名刺が見つからない

- 日付フィルタを見直してください。
- Eight 側で対象の名刺が登録済みか確認してください。

### 下書きが作成されない

- Outlook 認証が完了しているか確認してください。
- 連絡先にメールアドレスが入っているか確認してください。

## 11. 注意事項

- 必ず自分のユーザー名でログインしてください。
- 必ず自分の Microsoft アカウントで Outlook 認証してください。
- Eight セッションファイルは他人に渡さないでください。
